# pychord
A library to handle musical chords in python.

## Installation

```sh
$ pip install pychord
```

## Usage

```python
>>> from pychord import Chord
>>> c = Chord("Am7")
>>> print c.info()
Am7
root=A
quality=m7
appended=[]
on=None
```
